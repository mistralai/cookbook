# Flappy Bird

A self-contained Flappy Bird clone built with vanilla HTML5 Canvas and JavaScript. No dependencies.

## How to Play

1. Open `index.html` in any modern browser.
2. Press **Space**, **Arrow Up**, **Click**, or **Tap** to flap.
3. Navigate through the pipes without hitting them or the ground.
4. Your best score is saved in `localStorage`.

## Features

- Gravity-based bird physics with rotation
- Randomly generated pipes
- Score tracking with persistent high score
- Animated bird with wing flapping
- Sky gradient, clouds, and ground
- Works on desktop and mobile (touch input)
